## Excel Manipulation in Python 
This system is used for extracting data from excel and this is developed using python.
The packages used are:
<ul>
  <li>tkinter  </li>
   <li>openpyxl(xlrd,xlwt)  </li>
   <li>shutil  </li>
   <li>os  </li>
   <li>csv  </li>
</ul>
Options provided are:
<ul>
 <li>Browse Files  </li>
   <li>View Files </li>
   <li>Exit  </li>
   <li>Read  </li>
</ul>
